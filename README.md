# Market Data Analysis: Money.pl Dataset

This project focuses on the processing and analysis of market data sourced from the Money.pl financial portal. The analysis covers data preparation, trend identification, and the generation of a comprehensive analytical report.

## Key Project Stages
* **Data Acquisition & Integration:** Handling and merging multiple CSV datasets containing financial market information.
* **Exploratory Data Analysis (EDA):** Identifying key market trends and anomalies through descriptive statistics.
* **Data Visualization:** Creating clear and informative plots to represent financial fluctuations and market dynamics.
* **Reproducible Reporting:** Using R Markdown to integrate code and analysis into a professional HTML output.

## Main Conclusions
* The analysis provided insights into specific market trends and their stability over the analyzed period.
* Automated data cleaning procedures significantly reduced the time required to prepare raw financial data for analysis.
* The project demonstrates the ability to translate raw web-sourced data into actionable financial insights.

## Technologies
* **Language:** R
* **Tools:** R Markdown
* **Libraries:** tidyverse (dplyr, ggplot2), readr
